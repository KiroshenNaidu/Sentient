# **App Name**: Sentient

## Core Features:

- Text Input: Accept text input through direct entry or file upload (txt, csv, json).
- Sentiment Classification: Perform multi-class sentiment analysis (positive, negative, neutral) using an NLP API (e.g., Hugging Face).
- Confidence Scoring: Provide confidence scores for each sentiment classification to indicate the certainty of the prediction.
- Keyword Extraction: Extract key phrases and words driving sentiment and display them to the user to understand which words were classified to produce which sentiment
- Batch Processing: Enable efficient batch processing of multiple text entries.
- Comparative Analysis: Compare sentiment distributions across different texts or data sources with visual tools.
- Explanation Tool: Provide insight into why the model associated specific sentiment scores using reasoning, by highlighting key portions of the text in question. The LLM will use the input text and keyword extraction to generate an explanation.
- Export Results: Allow exporting results in CSV, JSON, and PDF formats for further analysis.

## Style Guidelines:

- Primary color: Deep blue (#2E3192), symbolizing intelligence and reliability.
- Background color: Light blue (#D4E0F7), offering a calm and unobtrusive backdrop.
- Accent color: Warm orange (#E67700) for highlighting key insights and interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif font offering a modern, machined and objective aesthetic. It's suitable for both headlines and body text, offering a unified look.
- Use clear, professional icons to represent sentiment scores and actions.
- Implement a clean and intuitive dashboard layout, dividing text entry, analysis results, and visualizations into distinct sections.
- Use subtle animations to enhance user engagement during sentiment processing.
